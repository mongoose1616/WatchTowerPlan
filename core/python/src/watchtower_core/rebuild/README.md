# `watchtower_core.rebuild`

## Summary
Export-safe rebuild harness primitives for deterministic regeneration of derived JSON indexes and rendered Markdown views from stronger authority.

## Use It For
- defining reusable derived-surface rebuild targets
- building rendered markdown views from the governed rendered-surface registry
- reconciling expected rendered markdown against the current workspace state
- running deterministic dry-run or write rebuild passes
- coordinating mixed JSON and Markdown outputs without importing repo-local planning services

## Keep Out
- plan-specific rebuild target catalogs
- live planning output shaping
- CLI formatting or command registration
