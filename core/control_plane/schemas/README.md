# `core/control_plane/schemas`

## Description
`This directory contains JSON Schemas for governed control-plane artifacts. Keep reusable definitions in common/, core-owned artifact schemas in artifacts/, and published external validation interfaces in interfaces/.`

## Paths
| Path | Description |
|---|---|
| `core/control_plane/schemas/README.md` | Describes the purpose of the schemas directory and its main schema families. |
| `core/control_plane/schemas/common/` | Shared schema definitions such as ids, references, timestamps, and enum-bearing fragments. |
| `core/control_plane/schemas/artifacts/` | Schemas for core-owned artifacts such as registries, policies, indexes, and contracts. |
| `core/control_plane/schemas/interfaces/` | Schemas external content must satisfy when validated by core. |
