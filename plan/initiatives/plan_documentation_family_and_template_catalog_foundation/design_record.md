# Plan Documentation Family And Template Catalog Foundation Design Record

## Summary
Adds the missing documentation-family registry and template catalog foundations for live plan guidance families and current rendered plan/project surfaces.

## Initial Design Boundary
- The initiative package is machine-first and local to `plan/initiatives/plan_documentation_family_and_template_catalog_foundation/.wt/`.
- Authored intake docs remain editable inputs but require explicit machine confirmation.
- Readiness must fail closed on missing capture, blocking deferred items, open discrepancies, or missing approval.
