# Plan Project Context Runtime Standardization Design Record

## Summary
Promotes project-scoped context loading into a first-class runtime surface layered on always-loaded pack context and proves it through a narrow query path.

## Initial Design Boundary
- The initiative package is machine-first and local to `plan/initiatives/plan_project_context_runtime_standardization/.wt/`.
- Authored intake docs remain editable inputs but require explicit machine confirmation.
- Readiness must fail closed on missing capture, blocking deferred items, open discrepancies, or missing approval.
