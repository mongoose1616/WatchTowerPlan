# Plan Duplicated Foundations Corpus Bootstrap Design Record

## Summary
Duplicates the current foundations corpus into core/docs/foundations and plan/docs/foundations so both roots carry the same guiding context required by requirements.md.

## Initial Design Boundary
- The initiative package is machine-first and local to `plan/initiatives/plan_duplicated_foundations_corpus_bootstrap/.wt/`.
- Authored intake docs remain editable inputs but require explicit machine confirmation.
- Readiness must fail closed on missing capture, blocking deferred items, open discrepancies, or missing approval.
