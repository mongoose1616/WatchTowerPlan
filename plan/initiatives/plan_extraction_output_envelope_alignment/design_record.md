# Plan Extraction Output Envelope Alignment Design Record

## Summary
Adds a typed extraction-output envelope helper and aligns guidance promotion with an explicit extraction stage instead of schema-only coverage.

## Initial Design Boundary
- The initiative package is machine-first and local to `plan/initiatives/plan_extraction_output_envelope_alignment/.wt/`.
- Authored intake docs remain editable inputs but require explicit machine confirmation.
- Readiness must fail closed on missing capture, blocking deferred items, open discrepancies, or missing approval.
