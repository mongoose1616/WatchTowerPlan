# Plan Artifact Family Registry Foundation Design Record

## Summary
Adds the missing artifact family registry so plan-pack artifact placement, status subsets, and derived index participation stop living only in path conventions and scattered code.

## Initial Design Boundary
- The initiative package is machine-first and local to `plan/initiatives/plan_artifact_family_registry_foundation/.wt/`.
- Authored intake docs remain editable inputs but require explicit machine confirmation.
- Readiness must fail closed on missing capture, blocking deferred items, open discrepancies, or missing approval.
