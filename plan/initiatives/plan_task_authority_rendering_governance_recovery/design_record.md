# Plan Task Authority, Rendering, and Governance Recovery Design Record

## Summary
Cuts over live task authority, restores human-readable planning renders, rewrites governance guidance, and validates the repaired operational flows.

## Initial Design Boundary
- The initiative package is machine-first and local to `plan/initiatives/plan_task_authority_rendering_governance_recovery/.wt/`.
- Authored intake docs remain editable inputs but require explicit machine confirmation.
- Readiness must fail closed on missing capture, blocking deferred items, open discrepancies, or missing approval.
